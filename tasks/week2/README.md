背景恢复方法说明：
- 1、首先遍历整个视频的每一帧，取像素平均值求得一副粗略还原的图像作为参考。
- 2、将该平均值的图像切割成左半边和右半边样本。并在中间切割一个宽度为80像素的中心样本。
- 3、将当前帧的中心样本与平均值的中心样本比较。若遮挡物在中间，舍弃该帧。
- 4、若遮挡物不在中间，判断遮挡物在左或右，而另外一边则为背景。
- 5、两边背景都获取后，拼接，则为最后的还原背景。
