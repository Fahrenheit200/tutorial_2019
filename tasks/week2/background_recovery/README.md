## Background Recovery

- Author: JiLi Liao
- Revised: JiLi Liao
- Date: 2019-09-29
- Version: 1.0.0
- Abstract: Recovering background from video.

### Depend Libraries
```
<chrono>
<vector>
<string>
<ctime>
```

### How to compile
``` shell
$ cd background_recovery
$ make
```

### How to run
``` shell
$ ./bin/demo ./resource/video0.mp4
```

### Sample Output
``` shell
$ Process time: 1.423s
# the result: ./result/res0.jpg
```
