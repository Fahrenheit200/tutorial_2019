## Title Here: findLongestCommon

- Author: JiLi Liao
- Revised: JiLi Liao
- Date: 2019-09-16
- Version: 1.0.0
- Abstract: A C++ program to find the lonest common suffix of some strings.

### How to run?

```shell
$ g++ findLongestCommon.cpp -o findLongestCommon.out
$ ./findLongestCommon.out
```

### Sample Input
```
ppp
pplpp
lpp
```

### Sample Output
```
pp
```
