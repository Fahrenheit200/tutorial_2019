## 第一周题目 —— 最大公共后缀

### Description

编写一个函数来查找字符串数组中的最长公共后缀。

如果不存在公共前缀，返回空字符串 ""。

### Input
编写一个函数，接口如下:
```
string longestCommon(vector<string>& strs) {

}
```

### Output
返回最长的公共后缀

### Sample Input
```
ppp
pplpp
lpp
```

### Sample Output
```
pp
```
