2. 在Ubuntu上用gedit编辑了一个c++程序，并编译运行
	在本地工作目录下，在终端输入：
	touch hello.cpp
	新建一个.cpp文件，再输入：
	gedit hello.cpp
	用gedit打开这个文件进行编辑。将有关代码编辑完成后保存，退出gedit。
	在终端输入
	g++ hello.cpp -o hello
	用g++编译该.cpp文件，生成一个可执行文件，再在终端输入：
	./hello
	运行该文件，输出hello world！
